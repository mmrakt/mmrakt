## 自己PR


### 自走力、キャッチアップスキル

自ら行動したりキャッチアップすることが得意です。現職のサイトは古い実装が多く、テストや仕様書も殆ど存在しなかったため改修コストや事故リスクが問題視されていました。そこで率先してリバースエンジニアリングと仕様のドキュメント化を3ヶ月間継続した結果、改修コストが前年比で約20%削減できました。
また別チームではサイトのフルリプレイスプロジェクトが進行し、段階的にリリースされた機能は私の所属チームで保守を行なっていました。その後十分な引き継ぎがないまま別チームは解散したため、所属チーム内での改修コストが問題化していました。そこで私はリプレイス後のアーキテクチャや技術スタックについて関連書籍やドキュメントから学び、チーム内でコードリーディング会を企画し2ヶ月間継続した結果、納期遅延やバグの発生率を低下させる事が出来ました。

### フロントエンド、バックエンド両方の開発経験

現職では主にシステムのバックエンドの実装を担当してきましたが、業務外では主にフロントエンドの設計・開発を行っています。言語はJavaScriptやTypeScript、フレームワークはReact.jsを用いたSPA開発が得意です。またNext.jsを用いたSSRやSSGの機能を搭載したアプリ、Gatsbyを用いたJamstackアプリの開発の経験があります。そのためシステムのインフラ領域を除く、広い領域での開発に対応することが可能です。
